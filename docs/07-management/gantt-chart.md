# Календарный план (диаграмма Ганта)

Планирование по 18-недельному графику МУ (старт — 02.03.2026). Диаграмма рендерится на
GitHub (Mermaid).

```mermaid
gantt
    title ИС «Автосалон» — календарный план (18 недель)
    dateFormat YYYY-MM-DD
    axisFormat %d.%m
    excludes weekends

    section Этап 0. Инициация
    Бизнес-анализ (паспорт, IDEF0, BUC, SWOT, ROI) :done, e0, 2026-03-02, 14d

    section Этап 1. Требования
    Use Case, Domain Model, трассировка           :done, e1, after e0, 14d

    section Этап 2. Архитектура
    PCMEF: пакеты, интерфейсы, ADR                :done, e2, after e1, 14d

    section Этап 3. База данных
    ER, DDL (3НФ), ORM-маппинг                    :done, e3, after e2, 14d

    section Этап 4. Детальное проектирование
    Sequence, классы, методы                      :done, e4, after e3, 14d

    section Этап 5. Реализация ядра
    Сервер (PCMEF) + клиент + оплата + тесты      :done, e5, after e4, 14d

    section Этап 6. Рефакторинг
    Паттерны, статический анализ, качество        :done, e6, after e5, 14d

    section Этап 7. Интерфейс
    Доработка UI, сценарии, оплата                :active, e7, after e6, 14d

    section Этап 8. Завершение
    Документация, ТЗ, записка, защита             :e8, after e7, 14d
```

## Вехи (Milestones)

- **M1** (нед. 4): утверждена модель требований.
- **M2** (нед. 8): спроектирована БД (3НФ), готов DDL.
- **M3** (нед. 12): работает ядро (сервер + клиент), проходят тесты.
- **M4** (нед. 16): завершён интерфейс и ключевые сценарии (включая оплату).
- **M5** (нед. 18): готов полный комплект документации, проект к защите.
