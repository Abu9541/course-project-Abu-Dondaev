# Документация проекта «ИС Автосалон»

Эта папка содержит проектную документацию курсового проекта в соответствии с
методическими указаниями (траектория **В — мобильная разработка**, PCMEF).

## Структура

| Папка / файл | Содержание |
|--------------|-----------|
| `01-business/` | Этап 0: бизнес-анализ (паспорт проекта, IDEF0, SWOT, BUC, стейкхолдеры) |
| `02-requirements/` | Этап 1: Use Case, Domain Model, глоссарий, трассировка |
| `03-architecture/` | Этап 2: диаграмма пакетов PCMEF, интерфейсы слоёв, ADR |
| `04-database/` | Этап 3: ER-диаграмма, DDL, стратегия ORM |
| `05-detailed-design/` | Этап 4: диаграммы последовательности, классы проектирования |
| `images/` | Диаграммы и скриншоты (PNG/SVG) |

> Приложение (исходный код БД, бэкенда и Android-клиента) находится в папке `../application`.
