# Изображения документации

Папка для скриншотов и экспортированных диаграмм, на которые ссылается документация
(относительными ссылками вида `![описание](images/file.png)`).

Большинство диаграмм в проекте выполнено на **Mermaid** и отображается прямо в `.md`-файлах
на GitHub — их экспортировать не требуется. Сюда добавляются преимущественно **скриншоты**,
которые нельзя сгенерировать из текста.

## Что разместить здесь при сдаче

| Файл | Назначение |
|------|-----------|
| `git-commit-activity.png` | График активности коммитов (GitHub Insights) — для README |
| `git-punch-card.png` | Тепловая карта коммитов по дням/часам (GitHub Insights) |
| `jacoco-summary.png` | Сводка покрытия тестами (отчёт JaCoCo) |
| `android-lint.png` / `sonar-overview.png` | Сводка статического анализа |
| `app-*.png` | Скриншоты экранов приложения (каталог, карточка, оплата, заявки и т. д.) |
| `swagger-ui.png` | Документация REST API (Swagger UI) |

> Раздел «Статистика разработки» в корневом `README.md` ссылается на `git-commit-activity.png`
> и `git-punch-card.png` — их нужно добавить на момент сдачи (см. требования МУ, п. 1.2.3).
