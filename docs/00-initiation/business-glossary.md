# Бизнес-глоссарий

Единый словарь терминов предметной области «Автосалон» (бизнес-уровень). Используется
во всех документах проекта для однозначного понимания.

| № | Термин | Определение |
|---|--------|-------------|
| 1 | **Автосалон** | Организация, осуществляющая розничную продажу новых автомобилей и сопутствующие услуги (тест-драйв, рассрочка). |
| 2 | **Автомобиль** | Конкретная товарная единица с уникальным VIN, характеристиками, ценой и статусом наличия. |
| 3 | **Марка (бренд)** | Производитель автомобилей (BMW, Toyota и т. п.), к которому относится модель. |
| 4 | **VIN** | Уникальный 17-значный идентификационный номер автомобиля. |
| 5 | **Каталог** | Совокупность доступных к продаже автомобилей с возможностью поиска и фильтрации. |
| 6 | **Комплектация** | Уровень оснащения автомобиля (например, «M Sport», «Prestige»). |
| 7 | **Клиент** | Физическое лицо, заинтересованное в подборе и покупке автомобиля. |
| 8 | **Менеджер по продажам** | Сотрудник салона, обрабатывающий заявки и заказы, сопровождающий сделку. |
| 9 | **Администратор** | Сотрудник, управляющий каталогом и учётными записями пользователей. |
| 10 | **Тест-драйв** | Пробная поездка на автомобиле, организуемая по предварительной записи. |
| 11 | **Дилерский центр** | Физическая площадка салона, где проводится тест-драйв и выдача авто. |
| 12 | **Заказ** | Оформленное намерение клиента приобрести конкретный автомобиль. |
| 13 | **Полная оплата** | Способ покупки, при котором стоимость автомобиля вносится единовременно. |
| 14 | **Рассрочка** | Способ покупки с первоначальным взносом и последующими ежемесячными платежами. |
| 15 | **Первоначальный взнос** | Часть стоимости, вносимая клиентом при оформлении рассрочки. |
| 16 | **Аннуитетный платёж** | Равный по сумме ежемесячный платёж по рассрочке. |
| 17 | **Платёж** | Финансовая операция оплаты заказа (в системе — имитация банковского эквайринга). |
| 18 | **Резервирование** | Перевод автомобиля в статус «забронирован» на время оформления заказа. |
| 19 | **Избранное** | Персональный список автомобилей, отмеченных клиентом для быстрого доступа. |
| 20 | **Отзыв** | Оценка (1–5) и комментарий клиента об автомобиле. |
| 21 | **Уведомление** | Сообщение пользователю о значимом событии (подтверждение, оплата, новая заявка). |
| 22 | **Заявка** | Обобщённое обращение клиента (запись на тест-драйв или заказ), требующее обработки персоналом. |
| 23 | **Воронка продаж** | Последовательность статусов от интереса клиента до завершённой сделки. |
| 24 | **Лояльность** | Уровень клиента (STANDARD/SILVER/GOLD/PLATINUM), отражающий историю взаимодействия. |

> Расширенный (системный) глоссарий с техническими терминами — см.
